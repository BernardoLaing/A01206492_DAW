¿Qué es entonces una base de datos - DB?

Es un conjunto de archivos (tablas) que guardan datos, en cada tabla de un tipo de registro.

¿En qué casos es conveniente usar bases de datos?

Cuando se tiene una cantidad grande de registros sobre los que el sistema necesita accesar sus datos, y los archivos se tienen que poder accesar por varios usuarios o aplicaciones, de esta manera se puede centralizar la información.

¿Qué es un sistema de gestión de bases de datos - DBMS?

Es un programa, o conjunto de programas, que permite extraer y almacenar información de una base de datos de manera conveniente y eficiente. Funciona como la interfaz entre el usuario y la base de datos.

Enlista y explica las funciones/responsabilidades que tiene un DBMS

  • Definición de los datos
  • Manipulación de los datos
  • Seguridad e integridad de los datos
  • Recuperación y restauración de los datos

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

En el proyecto de Fundamentos de Diseño Interactivo que realicé, habría sido bueno tener una base de datos. El proyecto era hacer un catálogo en línea de estudios de tatuajes, y tatuadores. Se tenía un listado de estudios con los tatuadores que trabajan en cada uno, así como con información de cada estudio, e información de los tatuadores, como una pequeña descripción y los trabajos que han realizado. En esto habría sido conveniente tener una base de datos para relacionar los estudios(entidad) con los tatuadores(entidad). Habría hecho que el proyecto fuera más escalable, ya que sería más fácil administrar el contenido.
